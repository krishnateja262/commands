# commands

1. Scala Version Number   
  ```
  scala> util.Properties.versionNumberString
  ```
2. Sbt version  
  ```
  $ sbt sbtVersion
  ```
3. Sbt continous build and run  
  ```
  $ sbt "~run"
  ```
4. Sbt show and inspect tasks  
  ```
  $ sbt "show hello"
  ```   
  ```
  $ sbt "inspect hello"
  ```
